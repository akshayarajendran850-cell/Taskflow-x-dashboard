TaskFlow X Dashboard is a modern web-based productivity application designed to manage daily tasks efficiently.
It allows users to add, edit, delete, and mark tasks as completed in a simple interface.
The application includes priority levels to organize tasks based on importance.
A dynamic progress bar shows real-time completion status of tasks.
Users can search tasks instantly using the search feature.
It also includes a basic login interface for user interaction.
Data is stored using LocalStorage, so tasks remain saved after refreshing the page.
The UI is designed with a dashboard layout including sidebar navigation and analytics cards.
Interactive elements like animations, icons, and hover effects enhance user experience.
This project demonstrates frontend development skills using HTML, CSS, and JavaScript
